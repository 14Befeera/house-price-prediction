#  House Price Prediction

This project predicts house prices using machine learning.

##  Features
- Predict house price based on input features
- Uses Linear Regression model
- Interactive UI using Streamlit
- Converts price into Indian Rupees

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

##  How it Works
- Data is loaded and preprocessed
- Features (X) and target (y) are separated
- Model is trained using Linear Regression
- User inputs are taken through UI
- Model predicts price and converts to INR

##  Example
Input:

Rooms: 6
LSTAT: 10
PTRATIO: 18

Output:

Predicted Price in INR
